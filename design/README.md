## 设计方向

### 色彩方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 背景 | 深空蓝 | `#0f0f23` → `#1a1a3e` |
| 卡片/面板 | 深紫蓝 | `#1e1b4b` |
| 主强调色 | 金色 | `#fbbf24` → `#f59e0b` |
| 起点格 | 翠绿 | `#10b981` → `#059669` |
| 奖励格 | 靛蓝 | `#6366f1` → `#4f46e5` |
| 冰冻格 | 天蓝 | `#38bdf8` → `#0ea5e9` |
| 炸弹格 | 红色 | `#ef4444` → `#dc2626` |
| VIP 格 | 金色 | `#fbbf24` → `#d97706` |
| 电池格 | 紫色 | `#a78bfa` → `#7c3aed` |
| 普通格 | 暗紫 | `#25224d` |

### 棋盘设计

- **布局**: 蛇形回廊式 (perimeter loop)，格子沿四条边排列
- **拐角**: 四角为转弯标记（虚线 + 折返箭头）
- **中心**: 骰子按钮 + 状态面板
- **格子样式**: 圆角卡片，类型图标 + 标签文字
- **玩家标记**: 金色光环高亮当前格 + 小车 emoji

### 排版

- 字体: Noto Sans SC（中文） / Inter（英文数字）
- 字号层级: 标题 18-20px / 正文 14-15px / 辅助 10-12px

### 响应式

- PC (>980px): 双列布局，左棋盘右面板
- 平板 (600-980px): 单列布局，面板横排
- 手机 (<600px): 单列纵向滚动，底部固定导航栏
