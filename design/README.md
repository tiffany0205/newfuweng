### 棋盘设计

- **布局**: 环形回廊式 (perimeter loop)，36 格沿四条边排列，4 个拐角转折
- **中心**: LUCKY JUMP 标题 + 骰子按钮（金橙渐变）
- **格子样式**: 深紫底色，类型图标 + 渐变彩色标签
- **玩家标记**: 金色高亮当前格
- **底色**: `#0a0a1a` 深空黑蓝，卡片 `#1e1b4b`

### 排版

- 字体: Noto Sans SC（中文） / Inter（英文数字）
- 字号层级: 标题 18-22px / 正文 14-15px / 辅助 10-12px

### 响应式

- PC (>980px): 双列布局，左棋盘 760x635px，右面板 570px 宽
- 手机 (<600px): 单列纵向滚动，紧凑状态栏，迷你棋盘，底部固定导航栏
